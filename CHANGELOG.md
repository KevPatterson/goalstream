## [0.1.0] - 2026-04-24

### ⚠️ Nota Importante
Este es un proyecto en desarrollo activo. Puede contener errores o funcionalidades incompletas.

### ✨ Añadido
- Página principal con grid de partidos
- Filtros por estado (en vivo, próximos, finalizados)
- Partido destacado con hero banner
- Reproductor de video HLS con controles personalizados
- Selector de calidad de video (Auto, SD, HD, FHD, 4K)
- Selector de streams por idioma
- Entrada manual de URL de stream
- Panel de administración para gestión de partidos
- Panel de administración para gestión de streams
- Detalles de partido en tiempo real
- Sidebar con lista de partidos
- Header con navegación responsive
- Menú móvil con animaciones
- Tema terminal retro con colores verde neón
- Efectos de escaneo y animaciones de pulso
- Soporte para Picture-in-Picture
- Diseño responsive para móviles y tablets
- Documentación completa en README.md
- Guía de contribución (CONTRIBUTING.md)
- Licencia MIT

### 🎨 Diseño
- Sistema de diseño basado en Tailwind CSS
- Paleta de colores personalizada (verde neón)
- Tipografía personalizada (Bebas Neue, IBM Plex Sans/Mono)
- Componentes UI reutilizables (Icon, Image, Logo)
- Animaciones y transiciones suaves
- Scrollbar personalizado
- Efectos de brillo (glow) en elementos activos

### 🛠️ Técnico
- Next.js 15 con App Router
- React 19 con hooks
- TypeScript para tipado estático
- HLS.js para streaming de video
- Configuración de ESLint y Prettier
- Configuración de Tailwind CSS
- Optimización de imágenes con Next/Image
- Variables CSS para tematización

### 📝 Documentación
- README completo con instrucciones de instalación
- Guía de contribución detallada
- Estructura del proyecto documentada
- Comentarios en código complejo
- Variables de entorno documentadas (.env.example)
